# Advanced Pomodoro Focus Dashboard

An advanced, high-precision productivity timer built using HTML, CSS, and JavaScript. The application utilizes system timestamp offsets to track time down to fractions of a second dynamically.

## 🚀 Features
* **Multi-Mode Engine:** Instantly toggle between Work sessions, Short Breaks, and Long Breaks.
* **High-Precision Clock:** Custom logic tracking centiseconds smoothly.
* **Automated Cycle Logic:** Tracks completed sessions and automatically prompts long break windows.
* **Configurable Settings:** Custom built-in modal UI to update session durations instantly.
* **Dynamic Themes:** Background styles shift color schemes fluidly based on the active mode.

## 🛠️ Built With
* HTML5
* CSS3 (Custom Grid, Flexbox, & CSS Variables)
* JavaScript (ES6+ Architecture)
